# Skills Database

## Goal

Stores all techniques and charms available in Sword of Convallaria.

---

## Structure

Each skill contains:

- ID
- Name
- Category (Technique / Charm)
- Rarity
- Icon
- Properties
- Tags
- Notes

---

## Properties

Properties may differ between skills.

Examples:

- Element
- Damage Type
- Target Type
- Cooldown
- Range
- AoE Radius
- Hits
- Effect
- Trigger Chance
- Duration
- Heal
- Shield
- Stats
